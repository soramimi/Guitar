# Guitar

**Guitar** is a fast, lightweight Git GUI client written in C++ with Qt 6.  
It provides an intuitive graphical interface for everyday Git workflows on Windows, macOS, and Linux.

## Features

### Core Git Operations
- **Commit** вЂ” Stage/unstage files, write commit messages, and commit with a single click
- **Branch** вЂ” Create, checkout, rename, and delete local/remote branches
- **Push / Pull / Fetch** вЂ” Sync with remote repositories; supports `--prune` and push all tags
- **Merge / Rebase** вЂ” Merge branches or rebase with abort support
- **Cherry-pick** вЂ” Apply individual commits across branches
- **Stash** вЂ” Stash and restore work in progress
- **Reset** вЂ” Soft and hard reset (`HEAD~1`, `--hard`)
- **Clone** вЂ” Clone repositories from URLs or directly from GitHub

### Viewing & Navigation
- **Commit log with graph** вЂ” Visual branch/merge graph with author avatars
- **Diff view** вЂ” Side-by-side and inline diff display with syntax-aware highlighting
- **Blame** вЂ” Line-by-line blame view with commit details
- **File history** вЂ” Per-file commit history browsing
- **Reflog** вЂ” Full reflog viewer for recovery
- **Find commit** вЂ” Incremental search across commit messages and file contents
- **Jump** вЂ” Quick navigation to any commit by hash

### Repository Management
- **Multiple repositories** вЂ” Manage many working copies from a unified sidebar
- **Working folders** вЂ” Organize repositories into named groups
- **Submodules** вЂ” Add and update Git submodules
- **Tags** вЂ” Create, edit, and push tags
- **Remotes** вЂ” Add, edit, and remove remote entries
- **Remote branches** вЂ” Browse and delete remote branches

### Advanced
- **GitHub integration** вЂ” Clone and search repositories from GitHub directly
- **SSH support** вЂ” Connect to remote hosts via SSH with key management
- **GPG signing** вЂ” Sign commits with a GPG key
- **AI commit messages** вЂ” Generate commit messages using a generative AI backend
- **Dark theme** вЂ” Full dark mode support
- **Multilingual UI** вЂ” English, Japanese, Spanish, Russian, Tamil, Chinese (Simplified/Traditional)
- **Credential helper config** вЂ” Manage Git credential helpers from the settings dialog
- **Trace logger** вЂ” Built-in trace/remote logging for debugging
