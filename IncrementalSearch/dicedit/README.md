# MeCab辞書エディタ

MeCab用ユーザー辞書ファイル（EUC-JP形式のCSV）をブラウザ上で編集するためのウェブアプリケーションです。

## 機能

- `dics/` ディレクトリ内の辞書CSVファイルを一覧表示し、編集対象を選択
- 1ページ25件のページナビゲーション（大規模辞書にも対応）
- セルをクリックして直接編集（フォーカスを外すと自動保存）
- 行の追加・削除
- キーワードによるリアルタイム検索（サーバー側フィルタリング）
- EUC-JPエンコーディングのままCSVファイルに上書き保存
- 保存後は辞書選択画面に戻る

## ディレクトリ構成

```
.
├── main.go              # サーバー本体
├── dics/                # 辞書CSVファイル置き場（EUC-JP形式）
├── templates/
│   ├── index.html       # 辞書選択画面
│   └── edit.html        # 辞書編集画面
├── static/
│   ├── css/style.css    # スタイルシート
│   └── js/app.js        # フロントエンドJS
├── go.mod
└── Makefile
```

## 辞書ファイルの形式

MeCab IPAdic互換のCSV形式（EUC-JP）を対象としています。

| 列 | 内容 |
|----|------|
| 0 | 表層形 |
| 1 | 左文脈ID |
| 2 | 右文脈ID |
| 3 | コスト |
| 4 | 品詞 |
| 5 | 品詞細分類1 |
| 6 | 品詞細分類2 |
| 7 | 品詞細分類3 |
| 8 | 活用型 |
| 9 | 活用形 |
| 10 | 原形 |
| 11 | 読み |
| 12 | 発音 |

## 使い方

### ビルドと起動

```bash
make run
```

または

```bash
go build -o dicedit .
./dicedit
```

ブラウザで `http://localhost:1080` にアクセスしてください。

### コマンド

| コマンド | 内容 |
|----------|------|
| `make build` | バイナリをビルド |
| `make run` | ビルドして起動 |
| `make clean` | バイナリを削除 |

## 動作要件

- Go 1.16以上（`//go:embed` を使用）
- 依存ライブラリ: `golang.org/x/text`
